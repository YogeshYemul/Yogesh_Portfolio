# 🚀 Yogesh Yemul - Portfolio

A modern, fully responsive portfolio built with **React 18**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Beautifully animated with a mobile-first design and deployed on Vercel.

## ✨ Features

- 🎨 **Modern Design** - Yellow, black, and white color theme
- 📱 **Mobile-First** - Fully responsive on all devices
- ✨ **Smooth Animations** - Framer Motion animations throughout
- 🎯 **Interactive Elements** - Hover effects and transitions
- 🌐 **Fast Performance** - Built with Vite for optimal speed
- 📊 **SEO Optimized** - Proper meta tags and structure
- 🔄 **Live Deployable** - Ready for production deployment

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Package Manager**: npm/bun
- **Deployment**: Vercel

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or bun

### Setup

1. **Clone the repository**
```bash
git clone <your-repo-url>
cd Yogesh_Portfolio
```

2. **Install dependencies**
```bash
npm install
# or
bun install
```

3. **Add your profile photo**
- Place your profile photo as `image.png` in `src/assets/` folder

4. **Add your resume**
- Place your resume as `resume.pdf` in the `public/` folder

## 🚀 Development

Start the development server:
```bash
npm run dev
```

## 🏗️ Build

Build for production:
```bash
npm run build
```

## 🌐 Deployment to Vercel

### Quick Deploy (GitHub)
1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Click "Deploy" (Vercel auto-detects Vite)

### Deploy via Vercel CLI
```bash
npm install -g vercel
vercel
```

## 📝 Configuration Files

- `vercel.json` - Vercel deployment configuration
- `.vercelignore` - Files to exclude from deployment
- `.env.example` - Environment variables template

## 🎨 Customization

Edit components in `src/components/` to update content and styling.
